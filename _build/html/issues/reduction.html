Relations between problems#

One of the things that we are very interested in in computability theory is in the relation between problems.

It will turn out that all of the problems which we have seen so far

  1. Tiling

  2. \(3\times 3\) Matrix Mortality

  3. Post Correspondence Problem (PCP)

are equivalent in some appropriate sense. Saying what this “appropriate sense” is will be important.
Also the details concerning equivalence are not trivial.

Definition

Let \(A\) and \(B\) be problems.

\emph{\(A \leq_m B\)} if there is an ``intuitively computable’’ \(f: N\to N\)

(or \(f: words\to words\), or \(N\to N\), or something similar)

such that for all \(x\).

\[ \emph{\mbox{$x\in A \quadiff f(x) \in B$}} \]

Again, \(f\) has to be total and computable! We say that \(A\) is reducible to \(B\).

The idea is that \(B\) is ``at least as hard as \(A\)’’ in some sense . (However, this intuition will have to be corrected, as we’ll see.)

Proof. For the first part: given \(x\), to see if \(x\in A\) or not, apply the computable function \(f\), and then see if \(f(x)\in B\) or not.

The answer must be the same!

The second assertion follows from the first.

Example

Let’s consider the \emph{Modified PCP (MPCP)}: Given a finite set of dominoes as in the PCP,

and one special domino \(d_0\) called the \emph{start domino}

is there a Post sequence that \emph{starts with \(d_0\)}?

\

I am going to give an informal proof that $\(\mbox{the PCP and the MPCP are equivalent} \)$ The basic idea is to do two things:

  1. If someone gave us an algorithm to solve the MPCP,

use this to get an algorithm to solve the PCP.

  1. If someone gave us an algorithm to solve the PCP,

use this to get an algorithm to solve the MPCP.

MPCP \(\leq\) PCP

Given an input to the MPCP, say \((\DD,d_0)\), take two \emph{new symbols}, say \(*\) and \(\$\), and then make a new domino set \(f(\DD)\) in the following way:

  1. Add \(*\) after all letters in all top words in the original domino set.

  2. Add \(*\) before all letters in all bottom words in the original domino set.

  3. Add a new copy of the start domino, \(d_0\), but modify it by adding \(*\) after all letters in the top and all letters in the bottom, and also add \(*\) before the first letter of the top word.

This one is called the \emph{first special domino}.

  1. Add a new domino with top word \(\$\) and bottom word \(*\$\). This last one is called the \emph{second special domino}.

Note that we have a intuitively computable function \(f\) from pointed tile sets to tile sets.

That is, adding the \(*\) like this is completely algorithmic.

\

Another point: we don’t need to know whether the original pointed domino set

has a Post word or not in order to add the \(*\)s.

Let \(\DD\) be the domino set

 If $\DD$ has any Post sequence starting with $d_0$,
 then $f(\DD)$ has a Post sequence.
 
 And if $f(\DD)$ has a Post sequence,  then $\DD$ has one that starts with $d_0$.

For \emph{all} \(\DD\), not just the one in our example:

If \(\DD\) has any Post sequence starting with \(d_0\), then \(f(\DD, d_0)\) has a Post sequence.

And if \(f(\DD, d_0)\) has a Post sequence, then \(\DD\) has one that starts with \(d_0\).

The main fact:

a Post sequence for \(f(\DD)\) must start with the first special domino, since this is the only one whose top and bottom words start with the same symbol.

It also has to end with the second special domino.

Now let’s prove the main claim.

Next, we show that PCP \(\leq\) MPCP This one is part of the \emph{second} homework.

You need to define an intuitively computable function \(g\) from domino sets to ``pointed domino sets’’ with the property that for all domino sets \(\DD\):

  1. If \(\DD\) has a Post sequence, then \(g(\DD)\) has one also, and one that starts with the start domino of \(g(\DD)\).

  2. Conversely, if \(g(\DD)\) has a Post sequence that starts with its start domino, then \(\DD\) has a Post sequence.

\begin{alertblock}{meaning of: MPCP \(\leq\) PCP} There is a computable function \(f\) from pointed domino sets \((\DD,d_0)\) to domino sets such that for all pointed domino sets \((\DD,d_0)\): \begin{itemize} \item[\(\tri\)] If \(\DD\) has a Post sequence starting with \(d_0\), then \(f(\DD, d_0)\) has one also, \item[\(\tri\)] Conversely, if \(f(\DD,d_0)\) has a Post sequence, then \(\DD\) has a Post sequence starting with \(d_0\) \end{itemize} \end{alertblock}

\begin{alertblock}{meaning of: PCP \(\leq\) MPCP} There is a computable function \(g\) from domino sets to pointed domino sets such that for all domino sets \(\DD\): \begin{itemize} \item[\(\tri\)] If \(\DD\) has a Post sequence, then \(g(\DD)\) has one also, and one that starts with the start domino of \(g(\DD)\). \item[\(\tri\)] Conversely, if \(g(\DD)\) has a Post sequence that starts with its start domino, then \(\DD\) has a Post sequence. \end{itemize}

\end{alertblock}

Definition

In this class, a \emph{problem} is a set \(P\),

usually a \emph{subset} of some \emph{easily defined = computable}

set \(S\).

\

We call it a \emph{problem} because we want to know

whether there is an algorithm to tell

if a given element of \(S\) is in \(P\), or not;

or whether there is no such algorithm.

\end{definition}

Example

\(S = \) all polynomials in one variable with integer coefficients.

\(P = \) all such polynomials which happen to have an integer root.

Definition

A problem \(P\) is \emph{decidable}

if there is an algorithm which,

when given an element of \(S\), tells in finite time

whether or not that input belongs to \(P\), or not.

Example

\(S = \) all polynomials in one variable with integer coefficients.

\(P = \) all such polynomials which happen to have an integer root.

\

This \(P\) is decidable.

Example

\(S = \) all polynomials in 9 variables with integer coefficients.

\(P = \) all such polynomials which happen to have an integer root.

This set \(P\) is not decidable.

Definition

Let \(P\) be a problem (as a subset of \(S\)),

and \(Q\) a problem (as a subset of \(T\)).

\

We write [ P \leq Q ] if there is a computable \(f: S\to T\)

such that for all \(s\in S\), [ s\in P \quadiff f(s) \in Q ]

$S =$  finite domino sets with a start domino
 
 $P = MPCP$
 
 $T = $ finite domino sets
 
 $Q = PCP$
 
 
 We saw earlier that $MPCP \leq PCP$.

Proof. Let \(f\) be the computable function showing that \(P\leq Q\).

\

Given \(s\in S\), to tell whether or not \(s\in P\), apply \(f\), and ask whether or not \(f(s)\in Q\).

(By assumption, there is a computable procedure to answer this question.)

The same answer holds for the question of whether or not \(s\in P\).

This proves the first assertion, and then the second one follows.

Proposition 2

Let \(A\leq B\).

  1. If \(B\) is decidable, then so is \(A\).

  2. If \(A\) is undecidable, then so is \(B\).

Exercises#

Tiling#

In the problems below on tiling, you should use the notion of reduction that we are currently studying.

Exercise 15

The one dimensional tiling problem is the problem of deciding whether a tile set can tile the “positive \(x\)-axis” part of the quadrant (the bottom infinite row). Show that this problem is decidable by reducing to the problem of whether a finite graph has a cycle.

Exercise 16

The two-row tiling problem is the same problem, but we want to tile the bottom-most two infinite rows. Is this problem decidable or undecidable?

Exercise 17

What about the two-column tiling problem?

Roots of polynomials#

Exercise 18

Let \(X\) be the set of multi-variable polynomials \(p(x_1, \ldots, x_n)\) with integer coefficients which have a root consisting of positive integers, and let \(Y\) be the polynomials with integer coefficients which have a root consisting of negative integers. Show that \(X \leq Y\) and \(Y\leq X\).

Post correspondence problem#

Exercise 19

Assuming that \(PCP\) is undecidable, show that it is undecidable even if the alphabet has only two letters.

Matrix mortality#

Exercise 20

Let \(P_3\) be the matrix mortality problem for \(3\times 3\) matrices. Let \(P_2\) and \(P_4\) be defined similarly, for the \(2\times 2\) and \(4\times 4\) matrices. Show that \(P_2 \leq P_3 \leq P_4\). This is a straightforward algebraic argument.

Later we will see that \(P_4 \leq P_3\), but the argument will not be not as easy.

Here is an exercise pertaining to the matrix mortality problem. Athough it looks long, we are presenting it as result in outlined form. All you have to do is to fill in the steps. This result will be used in our later work on the undecidability of matrix mortality.

Exercise 21

Let \(M_{1,1}\) be the set of finite sets of \(3\times 3\) integer matrices which have some finite product which equals \(0\). Let \(M_{0}\) be the set of finite sets of \(3\times 3\) integer matrices which have some finite product with \(0\) in the upper-left corner. In this exercise, we show that \(M_{1,1} \leq M_0\).

Let \(A\) be the \(3\times 3\) matrix

\[\begin{split} \left[ \begin{array}{lll} 1 & 0 & 0 \\ 0 & 0 & 0 \\ 0 & 0 & 0 \end{array} \right] \end{split}\]

The key features of \(A\) are shown below:

\[\begin{split} \left[ \begin{array}{lll} 1 & 0 & 0 \\ 0 & 0 & 0 \\ 0 & 0 & 0 \end{array} \right] \left[ \begin{array}{lll} a & b & c \\ d & e & f \\ g & h & i \end{array}\right] %\left[ \begin{array}{lll} 1 & 0 & 0 \\ 0 & 0 & 0 \\ 0 & 0 & 0 \end{array} \right] = \left[\begin{array}{lll} a & b & c \\ 0 & 0 & 0 \\ 0 & 0 & 0 \end{array}\right] \end{split}\]

We also have

\[\begin{split} \left[ \begin{array}{lll} 1 & 0 & 0 \\ 0 & 0 & 0 \\ 0 & 0 & 0 \end{array} \right] \left[ \begin{array}{lll} a & b & c \\ d & e & f \\ g & h & i \end{array}\right] \left[ \begin{array}{lll} 1 & 0 & 0 \\ 0 & 0 & 0 \\ 0 & 0 & 0 \end{array} \right] = \left[\begin{array}{lll} a & 0 & 0 \\ 0 & 0 & 0 \\ 0 & 0 & 0 \end{array}\right] \end{split}\]

Let \(Q\) be the set of finite sets of \(3\times 3\) integer matrices.

Here is a computable function \(f: Q \to Q\):

\[ f(S) = S\cup \{A\} \]

Our goal is to prove the following fact:

Let \(S\) be a finite set of \(3\times 3\) matricies.

  1. Show that if \(S \in M_{1,1}\), then \(f(S) \in M_0\). [This is easy given the facts above.]

  2. In the other direction, suppose that \(f(S) \in M_0\). Take a finite sequence from \(f(S)\) whose product has a zero in the upper-left corner. Write the product as

\[ A B^1_1 B^1_2 \cdots B^1_{n_1} A B^2_1 B^2_2 \cdots B^2_{n_1}\cdots A B^k_1 B^k_2 \cdots B^k_{n_k} A \]

where the \(B^i_j\) matrices belong to \(S\).

Divide the product above into \(k+1\) groups:

\[ (A B^1_1 B^1_2 \cdots B^1_{n_1}) \quad (A B^2_1 B^2_2 \cdots B^2_{n_1}) \quad\cdots\quad (A B^k_1 B^k_2 \cdots B^k_{n_k}) \quad A \]

And then multiply it group-by-group:

\[\begin{split} \left[\begin{array}{lll} a^1 & b^1 & c^1 \\ 0 & 0 & 0 \\ 0 & 0 & 0 \end{array}\right] \left[\begin{array}{lll} a^2 & b^2 & c^2 \\ 0 & 0 & 0 \\ 0 & 0 & 0 \end{array}\right] \cdots \left[\begin{array}{lll} a^k & b^k & c^k \\ 0 & 0 & 0 \\ 0 & 0 & 0 \end{array}\right] \left[\begin{array}{lll} 1 & 0 & 0 \\ 0 & 0 & 0 \\ 0 & 0 & 0 \end{array}\right] \end{split}\]

Here each \(a^i\) is the product of the \(i\)th group.

Evaluate the upper-left corner of the product, and check that one of the groups \( B^i_1 B^i_2 \cdots B^i_{n_i}\) must multiply to give a matrix with \(0\) in the upper-left corner.
